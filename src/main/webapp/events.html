Events

Events are essential for Rosette. An event can be a church service, a prayer meeting etc.

  • Read
  • Create
  • Update
  • Delete

Read

Resource URL

http://api.ryttargarden.se/api/v1-snapshot/events

Parameters

since

Specifies the date for the first event to get.

Example value: 1335348000000
until

Specifies the date for the last event to get.

Example value: 1335348000000

Example Request

GET http://api.ryttargarden.se/api/v1-snapshot/events

Example Response

200 OK
  
[
    {
        "_id" : ObjectId( "4f96fb67d4c41b0e98447f81" ),
      "title" : "Sunday Service"
    },
    {
        "_id" : ObjectId( "4f96fb67d4c41b0e98447f81" ),
      "title" : "Prayer meeting"
    }
]
